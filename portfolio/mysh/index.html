Custom UNIX-Like Shell

View on GitHub
C Linux POSIX GNU Make GCC

Overview

  • Implemented a fully functional interactive shell in C, targeting Linux and adhering to standard POSIX conventions
  • Compiles cleanly with GCC; build system managed with GNU Make for efficient incremental compilation

Features

  • Pipe chaining and command sequencing, replicating the behavior of complex pipelines like grep | ls | cat
  • Full input and output redirection using fopen, read, and write system calls
  • Concurrent process execution via fork(), allowing background jobs and parallel command handling
  • Built-in command support alongside PATH-resolved external program execution
